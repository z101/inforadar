# Архитектурный дизайн приложения "Информационный радар"

## Summary

Документ содержит техническое описание приложения "Информационный радар", которое предcтавляет собой консольное интерактивное приложение, запускаемое в терминале WSL Windows.

## Общая структура приложения

Приложение состоит из CLI модуля на Python и отдельной базы данных на SQLite, в которой хранятся настройки приложения, информация по источникам, их разделам и статьям.

## Архитектура CLI модуля

CLI модуль полностью интерактивен и написан с на Python-библиотеке `rich`. В основу положена концепция экранов для различного вида функциональности.

## Запуск приложения

Запуск приложения организован через Makefile (`make` util). Доступные команды:

`make setup` - Создать виртуальное окружение и установить зависимости для разработки.
`make install` - Установить команду 'ir' глобально для текущего пользователя в WSL.
`make uninstall` - Удалить команду 'ir' глобально для текущего пользователя в WSL.
`make test` - Запустить тесты внутри виртуального окружения.
`make coverage` - Запустить тесты и показать отчет о покрытии кода.

Debug-режим (запуск из venv для отладки):

`make debug` - Запустить 'ir' (TUI) в режиме отладки (по умолчанию).
`make clean` - Удалить временные файлы и виртуальное окружение.
