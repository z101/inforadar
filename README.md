# Информационный радар (inforadar)

Консольное приложение для сбора и эффективного разбора технических статей из различных источников.

## MVP

MVP сфокусирован на сборе статей с Habr.

### Установка и запуск

1.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv .venv
    source .venv/bin/activate
    ```

2.  **Установите зависимости и приложение:**
    ```bash
    pip install -e .
    ```

3.  **Настройте конфигурацию:**
    Скопируйте `config.yml.example` в `config.yml` и отредактируйте список хабов и фильтры.

4.  **Используйте приложение:**
    ```bash
    # Собрать новые статьи
    ir fetch

    # Начать интерактивный разбор
    ir list --interactive
    ```
