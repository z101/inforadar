# План реализации MVP "Информационный радар"

**Итоговая оценка:** 5-6 рабочих дней.

---

### Этап 1: Фундамент и настройка (1 день)
1.  **Структура проекта:** Создать директории, инициализировать git.
2.  **Зависимости:** Установить `typer`, `rich`, `sqlalchemy`, `pyyaml`, `requests`, `beautifulsoup4`, `feedparser`, `webbrowser`.
3.  **Модель данных:** Определить SQLAlchemy-модель для таблицы `articles` (с `guid`, `status_read`, `status_interesting` и `metadata: JSON`).
4.  **Хранилище (`Storage`):** Реализовать класс-репозиторий с методами: `add_articles_if_not_exist`, `get_articles_by_status`, `update_article_status`.

### Этап 2: Ядро сбора данных (2 дня)
1.  **Конфигурация:** Написать код для чтения `config.yml`.
2.  **`HabrProvider`:** Создать основной класс.
3.  **Реализация Шага 1 (RSS):** Внутри провайдера написать логику получения статей из RSS-фидов с помощью `feedparser`.
4.  **Реализация Шага 2 (Скрапинг):** Написать метод `_enrich_article_data`, который принимает URL, скачивает HTML и парсит его с помощью `BeautifulSoup`.
5.  **Интеграция и фильтрация:** Связать оба шага. После обогащения применить фильтры из конфига. Реализовать "этичный" скрапинг с задержками.

### Этап 3: Пользовательский интерфейс (2 дня)
1.  **CLI-каркас:** Настроить `Typer` для создания команд.
2.  **Команда `fetch`:** Реализовать команду, которая запускает `HabrProvider`, показывает прогресс-бар (`rich.progress`) и сохраняет результат в `Storage`.
3.  **Команда `list --interactive`:** Самая сложная часть. Реализовать цикл `while`, который получает одну статью, отрисовывает "карточку" с помощью `rich`, ждет ввода пользователя (`input()`), обновляет статус статьи в `Storage` и переходит к следующей.
4.  **Вспомогательные команды:** Реализовать простой табличный `list` и `open <id>` (используя `webbrowser.open`).

### Этап 4: Финализация (0.5 дня)
1.  **Написание `README.md`:** Подробно описать, как установить зависимости, настроить `config.yml` и запустить приложение.
2.  **Тестирование:** Провести ручное тестирование основного сценария.
